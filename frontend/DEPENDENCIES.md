# Frontend Dependency Reference

## Runtime
- **react** / **react-dom** - UI library
- **react-router-dom** - Client-side routing
- **@tanstack/react-query** - Server-state caching and background refetch
- **axios** - HTTP client
- **zod** - Schema validation
- **react-hook-form** + **@hookform/resolvers** - Form state and Zod integration
- **framer-motion** - Animations
- **react-icons** - Icon set
- **recharts** - Dashboard charts
- **react-hot-toast** - Toast notifications

## Build / Dev
- **vite** - Dev server and bundler
- **@tailwindcss/vite** - Tailwind v4 integration
- **vite-plugin-pwa** - Service worker and manifest generation
- **typescript** - Type checking
